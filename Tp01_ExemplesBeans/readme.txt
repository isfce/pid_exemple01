Cet exemple montre les 3 techniques de configuration de beans:
Il existe 3 classes "main", une par type de configuration
Il existe 3 tests unitaires pour chaque configuration.

Vous pouvez exécuter les tests unitaires en allant dans 
"gradle tasks" et faisant un double clic sur le mot "test" (du projet)

Sinon pour les méthodes "main", vous pouvrez les excécuter individuellement
